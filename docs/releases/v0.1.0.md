# v0.1.0

Initial public release of **pmpp-cuda-study**, a CUDA study repository organized around a PMPP-style progression from simple kernels to advanced parallel workloads.

## Highlights

- Added **100 numbered CUDA study examples** spanning foundations, parallel patterns, linear algebra, image and signal processing, simulation, graph, and ML topics.
- Implemented **working CUDA code for examples `001-020`**, covering foundational kernels such as vector operations, image transforms, stencils, matrix addition, and matrix transpose.
- Added **guided CUDA templates for examples `021-100`** so the full learning path is present from day one.
- Added **per-example documentation** with goals, PMPP concepts to focus on, build/run steps, and follow-up ideas.
- Added a **top-level study index** and a **5-batch publishing plan** to support staged expansion and review.
- Added **CMake support** for CUDA-enabled environments.
- Added a repo-local **`.clang-format`** and performed a readability/style cleanup across all example source files.

## Repository Structure

- `examples/001-020`: reference-friendly implementations with validation logic
- `examples/021-100`: guided templates for continued implementation work
- `docs/`: conventions, publishing notes, and release notes
- `tools/`: helper scripts used during repo generation and cleanup

## Notes

- This release emphasizes **study value and structure** over claiming every advanced example is fully implemented.
- The first 20 examples are the most runnable and review-ready part of the repository.
- The remaining examples are intentionally scaffolded so the repository can grow in clear batches over time.
- `nvcc` was not available in the authoring environment used for this release, so CUDA compilation was not executed as part of this packaging step.
